# Swipeify

A Tinder-style music discovery app that lets you swipe through songs and build playlists you'll actually love.

---

## What is this?

Swipeify makes discovering new music feel effortless. Instead of scrolling through endless recommendations, you swipe right on songs you like and left on ones you don't. It connects to Spotify, so everything you save is ready to listen to right away.

---

## Features

- Swipe through song recommendations
- Swipe right to save a song, left to skip it
- Spotify integration for seamless playback
- Clean, native Android experience

---

## Tech Stack

**Android App**
- Kotlin
- Jetpack Compose
- Spotify API

**Backend**
- Python (FastAPI)
- SQLite

---

## Getting Started

### Backend

```bash
cd backend
pip install fastapi uvicorn
uvicorn main:app --reload
```

### Android App

Open the project in Android Studio and run it on an emulator or physical device. Make sure to add your Spotify API credentials before building.
